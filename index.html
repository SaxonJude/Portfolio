Utilising:

JS ES6

Axios

Git

HTML5

CSS3

Visit Project View Source

The Forkify Recipe App allows users to search for recipes using the Food2Fork API. Users can also view the recipe along with the cook time ingredients. Users can add their favorite meals to a favorite meals section which is stored using local storage so no need for a backend. Webpack was used as the module bundler to incorporate Axios for fetching. No frameworks were used in this application as it was coded in vanilla Javascript.
